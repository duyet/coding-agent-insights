# /cai tui

Launch interactive terminal UI for exploring coding history.

## Usage

```
/cai tui
```

## Features

- **Browse** - Navigate entries with keyboard
- **Filter** - Search and filter in real-time
- **Analyze** - View charts and statistics
- **Export** - Export filtered results

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑/↓` | Navigate entries |
| `/` | Search |
| `f` | Open filter panel |
| `s` | Sort options |
| `e` | Export current view |
| `q` | Quit |
